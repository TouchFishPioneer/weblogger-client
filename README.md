# web-logger-client

我他妈也很绝望啊，出国在外还被老板逼着写代码。为了表示我抗争的决心，这个 README 我将全文使用汉语编写。

## 项目简介

6316实验室的祖传项目，weblogger 的客户端部分的代码库。该库通过 JavaScript 采集在前台点按事件发生时移动设备惯性传感器记录下的传感器数据。

## 功能特性

我也不知道有啥特性。

```
vue-cli:      ^3.0.1
vue:          ^2.5.17
vue-router:   ^3.0.1

```

## 部署步骤

```
npm run build
```
